Alterando o tamanho do cabeçalho.

Para mudar o tamanho da fonte de um cabeçalho utilizamos as seguintes tags.

< h1> Até < h6>

Lembrando que quanto maior o número depois do "h" menor sera o tamanho!

Aqui temos exemplos


< h1>Cabeçalho < h1> aqui!
< h2>Cabeçalho < h2> Aqui!
< h3>Cabeçalho < h3> Aqui!
< h4>Cabeçalho < h4> Aqui!
< h5>Cabeçalho < h5> Aqui!
< h6>Cabeçalho < h6> Aqui!

Aqui temos o resultado

Cabeçalho < h1> aqui!

Cabeçalho < h2> Aqui!

Cabeçalho < h3> Aqui!

Cabeçalho < h4> Aqui!

Cabeçalho < h5> Aqui!
Cabeçalho < h6> Aqui!
