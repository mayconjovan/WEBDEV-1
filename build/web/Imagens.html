Inserir Imagens

Para Inserir imagens em página podemos utilizar algumas formas, porém todas elas chegam ao mesmo resultado.

Código para que isso aconteça:

src="pastaDaImagem/nomeDaImagem.formato"

também deve-se colocar um alt="" (para navegadores que não conseguirem abrir a imagem).

No caso da imagem abaixo o código fica assim:

< img src="image/must.jpg" alt="Foto de um mustang."/>
