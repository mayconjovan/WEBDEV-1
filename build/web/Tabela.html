Criando tabelas



Passo-a-passo como criar uma tabela na página.

Tabela de Tags
TAG Função
< table > Define que será desenhado uma tabela
< th > Define nome para uma coluna
< tr > Define o desenho da tabela
< td > Define o conteudo daquela célula
< /"tag" > O sinal < / Fecha a tag desejada
border Dentro da tag < table border="1"> define as bordas da tabela
< table disabled > Disabled, desativa a caixa de de texto para q não modifiquem.
< td colspam="2" > Define o numero (2) de colunas que farão a junção.
< td rowspam="2" > Define o numero (2) de linhas que farão a junção.
